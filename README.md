# yasnippet-purescript
Collection of PureScript snippets I use in Emacs everyday

## Installation

Clone this repo inside the `snippets` directory of your `.emacs.d` directory.

```
.emacs.d
 |
 |__ snippets
      |
      |__ purescript-mode
```
